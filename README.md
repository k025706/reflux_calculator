# 逆流性弁膜症 定量評価計算ツール

心エコー検査における弁膜症逆流（MR・AR・TR・PR）の重症度定量評価を、ベッドサイド／エコー室で即座に行うための単一HTMLツールです。ブラウザで開くだけで動作し、インストールやサーバーは不要です。

## 対象疾患と評価手法

| 疾患 | PISA法 | Volumetric法 |
|---|---|---|
| MR（僧帽弁閉鎖不全症） | ○ | ○（僧帽弁輪を楕円近似） |
| AR（大動脈弁閉鎖不全症） | ○ | ○ |
| TR（三尖弁閉鎖不全症） | ○（PISA半径の直接判定含む） | × |
| PR（肺動脈弁閉鎖不全症） | × | × |

- **PISA法**：PISA半径・Aliasing Velocity・Peak Velocity・Regurgitant VTIからEROA／RVolを算出
- **Volumetric法**：LVOT径・LVOT VTI・僧帽弁輪長径短径・弁輪VTI・Regurgitant VTIからRVol／RF／EROAを算出

## 主な機能

- 重症度カットオフ基準値の参照表示（定量評価は常時表示、心腔サイズ・定性評価は折りたたみ表示）
- EROA・RVol・RF・PISA半径直接判定について、Mild/Moderate/Severeの重症度基準を数直線で可視化
- 計測値間の重症度判定が乖離している場合の警告表示
- 「見た目の重症度に近づけるには、どのパラメータをどの程度変更する必要があるか」を示すシミュレーション機能（各指標が非Mildの場合、隣接する重症度に到達するための変更を提示）
- 生理学的にありえない値（負のRVolなど）に対する警告
- ダークモード（デフォルト）／ライトモード切替
- モバイル・タブレット表示に対応したレスポンシブデザイン

## 免責事項

- 本ツールが扱うのは心エコー図検査（TTE/TEE）による指標のみです。心臓MRI・心臓カテーテル検査による評価指標は対象外です。
- 定性的・半定量的評価（カラードプラの見た目の判定など）は参照表示のみであり、自動判定は行いません。
- 本ツールの計算結果は臨床判断を支援する参考情報であり、診断・治療方針の決定は必ず医師の総合的な判断によって行ってください。本ツールの使用によって生じたいかなる結果についても、作者は責任を負いません。

## 使い方

`index.html` をブラウザで開き、対象疾患を選択のうえ、各カードに計測値を入力してください。
